Variable Fonts

What is a Variable Font? A Variable font is a essentially one file to have your different font styles located. This makes it so you don't need two versions of the font bold and not bold, for example. This can allow your website to load quicker due to less font-styles needing to be loaded in. Variable Fonts can allow a website to have animations or transitioning font styles on the fly! Next will be the several differnt tags that can be used to alter your text.

Weight (which is the "wght" tag) allows the user to change how bold, or none bold they want their text to look like. The value can be changed between 300 - 900. In the example next though these values are only between 300-900. Examples are listed below.

Another way the font can be manipulated is changing the width or known through the "wdth" tag, which allows you to change how wide or thing text appears. The values can be changed from 55% to 100% in the Example below.

Italic text is simple to add! Text can be italicized with "ital" tag. However, with italic font the only values that can be set are 0-1. 0 simply means that the text wont be italic, while 1 means it will be italic. The values can be changed from -90 to 90 degrees. Example Below of this

Slant is somewhat similar to italic, however the difference between them is slant doesn't change any of the characters at all. Slant just simply changes the angle of the text. The tag to slant your text is the "slnt" tag. An example of Slant is next.

Grade is similar to weight, because it can be used to change how thick the text is. The tag for Grade is "Grad" and when the tag is used, the text can appear more bolded, however the actual physical space that the text takes up does not change. An example of the code is next.

One thing to keep in mind is not all browsers support all font sizes and stylings. These days it's not that much of an issue, but if using older browsers keep in mind that text may not display or load as intended.

If looking for different fonts to add to your project, a common source is from Google Fonts. This is what I used to find the font for my website!

Overall, variable fonts allow you to change your fonts easily, without much hassle. Without variable fonts designing projects would be more tedious and most importantly of all, would be more time-consuming. It has become almost an essential for the creation of projects.
